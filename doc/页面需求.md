# 蕴光工作室主页 — 页面需求文档

## 概述

蕴光工作室官网（yunlite.com），作为品牌门户和产品导航站。初期纯静态部署，后续随产品线扩展逐步完善。

## 当前版本 (v1.0)

### 技术方案
- **纯静态**: HTML + Tailwind CSS (CDN) + 少量 JS
- **零构建依赖**: 无 npm/node，直接 serve
- **部署**: Vercel（免费），后续可迁移到自有 Nginx
- **仓库**: github.com/raogang1999/yunlite

### 设计规范

| 项目 | 规范 |
|------|------|
| **风格** | 浅色温暖风，米白底 (#FAFAF8) |
| **强调色** | 暖金/琥珀 (#E5A91B / #C8860F) |
| **辅助色** | 鼠尾草绿 (#6E8E5D) — 成长感 |
